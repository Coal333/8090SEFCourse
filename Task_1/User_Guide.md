Task 1

In this project, a HKMU bank account management system will be developed. It does bank operations from the
bank admins' perspective. The program will do the tasks as shown below. The tasks may be elaborated upon later on. The program will offer a menu to the user and the user will have to choose an option from multiple options to proceeed to the commands.

User Guide:

Link to the 5 minute video:

Make sure to have Python 3.x installed

Run:
python main.py
(While you are in the Task_1 directory in powershell)

How to use the program:

Main menu would look like this:

    """Welcome to the HKMU Banking System"""
    Please select one of the options to proceed:

    1. Customer Management
    2. Exit Program

Choose Customer Management by entering the number 1

Enter the Customer ID, Example: A0000001

Then you will see this menu:

    Please select one of the options to proceed:
    1. View Customer Information
    2. Edit Customer Information
    3. Transfer Money
    4. Go back to the first page

Press 4 then press 1 again and enter B0000003

You will see a different menu:

    Please select one of the options to proceed:
    1. Access the Customer Management Menu
    2. Access the Contract Managment Menu
    3. Generate Today's random color

Press 1 and you will access the same menu as shown before

Below, is a following explanation on what each of these menu options do:

View Customer Information: View the Account's record data
Edit Customer Information: Edit the Account's record data
Transfer Money: Transfers money from one account to another account
Go back to the first page: Goes back to the main menu

Access the Customer Management Menu: Goes to the menu shown above
Access the Contract Managment Menu: Goes to the contract menu as shown below
Generate Today's random color: Generate today's random color

Below is the Contract Management Menu, it does the following

View all Contracts: View all contracts of the account
Create a new Contract: Create a new contract under the account
Go back to the first page: Goes back to the main menu
